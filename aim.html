User will have the ability to either input recipes themselves or have Bake Bot to search the web for recipes. User will need to input how many serves they would like to make. As user has input the number of serving. Bake Bot will be able to convert the measurements from the original recipe to match the number of serving needed. Bake Bot would then create a step-by-step recipe guide on how to cook the chosen recipe. Bake Bot will also use the original recipe that user has chosen and create the necessary timers for the recipe. This is the backbone of the app.

Bake Bot will also have the availability to favourite and creating a recipe journal.

Goal 2 - Purchasing Baking Courses

To help monetise the app we will be creating baking/cooking courses the user can purchase. To "wet the user's whistle" Bake Bot will provide a handful of baking basics to provided free.

Goal 3 - Purchase Products

Another way Bake Bot will be able to monetise the app is the ability to purchase products.

Goal 5 - Meal Planner

As user has created a journal of recipes. They will then be able to create a meal plan with the saved recipes

Goal 6 - groceries list

As a meal plan has been created a groceries list can be created. Having a groceries list is important as it help the user to stay on track and not be distracted by other items in the store. Resulting in money saved.

The products that are required will match the product at the store. This will also reduce food wastage.
