Roles

Business Manager - Peter

Bake Bot's Business Manager prepares and oversees the administration of the business. They will need to ensure the organisation meets all its goals. The Business Manager advises the management and stakeholders on the development of Bake Bot. The development of Bake Bot's business plan will be created by Bake Bot's Business Manager. Enable him to keep the rest of the team on track.

They will need to develop and build relationship with manufactures to develop Bake Bot branded baking requirement.

GUI Designer - Blake

The GUI designer will need to create the graphical user interface (GUI) for Bake Bot. This GUI will need to be user-friendly and convey the goals of Bake Bot. BakeBot’s designer will need to work with the front end of the application.

  

With the Software Engineer, the GUI Designer will need to make all the connections from the code to procedure a useable application. They are also in charge of making the application functional and visually pleasing to the user.

  

The GUI Designer will also need to protype the Bake Bot application with the assistance of the Software Engineer and the Software Tester.

Program Tester - William

Bot's Software Tester will work with the software engineer to make sure the application performs and functions as per the goals of Bake Bot. The Software Tester will need to be the end-user. The end-user is extremely important as they need to find any bugs, errors, defects or any other problem that will affect the functionality of the Bake Bot.

They will need to inform the software engineer or the GUI designer of the potential problems.

Marketing Manager - Josh

Bake Bot's Marketing Manager will work with the Business Manager and the Lead Design to present a user-friendly app. Building relationship with cooks and chefs to promote and develop online baking courses. The Marketing Manager will also need to find ways to promote the Bake Bot, online courses and baking products.

Creating a social media present is vital to the survival of Bake Bot as making people are consuming more content online. These campaigns will attract more user and turn them from free-user into paid-users. Bring in revenue to the Bake Bot.

Software Engineering - Martina

Bake Bot's software engineer will develop code that is needed for the Bake Bot application. This includes writing the backend code that will link recipes measurements to the measurement conversion calculator (front end user).

They will also develop the code that to reach the aims (goals) of the business. Software Engineer will need to work with the Lead Designer and the Program Tester to make sure all the aims of Bake Bot have been met.
